## Три блокера релиза iOS (подтверждены построчно)

1. **Вход в Client невозможен** — `Shared/Models.swift:264` требовал `expiresIn`,
   которого сервер не шлёт. Декодирование падало на каждом запросе кода. **Закрыто**
   (`93da1e9`), проверено на симуляторе.
2. **Курьер не мог закрыть доставку** — DTO без `evidenceIdempotencyKey` + метка
   `"delivery_proof"` вместо `"Подтверждение доставки"`. **Закрыто** (`9b18641`).
3. **Сброс PIN с locked-экрана без знания старого** — все восемь приложений.
   **Закрыто** (`e892091`).

## Как читать

Порядок исполнения внутри волн задан ролями и не всегда совпадает с порядком
таблиц (напр. волна 2: 2.1 → 2.3 → 2.4 → 2.2 → …, потому что версионирование
SwiftData сегодня денег не теряет, но превращает в потерю любой следующий срез).
Каждый срез несёт раздел «Опровергнуто» — что сделано правильно и не подлежит правке.

## Заблокировано на владельце

Ротация трёх ключей (`OPENROUTER_API_KEY`, токен туннеля Cloudflare, ключи App Store
Connect); Guideline 4.2 по трём B2B-приложениям; демо-аккаунты, App Privacy, цена и
контакт ревью; доступ к Mac mini под сервер. Без этого «магазин в сторе» недостижим
независимо от объёма кода.

# AliStore — Native Design System (iOS + Android + Web)

Подход: **гибрид** — бренд одинаковый на всех платформах, системные паттерны отдаём ОС.
Так делают Ozon, Wildberries, Kaspi. Прототипы `.dc.html` — единый визуальный эталон.

---

### Форма и ритм
- Радиусы: карточки 14–22, кнопки 10–13, чипы/бейджи 6–999
- Отступы: сетка 4pt (4/8/12/16/20/24)
- Карточка товара: фото сверху (радиус 12), название 2 строки, рейтинг ★, цена (Sora 800), кнопка «В корзину»
- Иконки категорий, логотип (оранжевый квадрат 10–14 радиус, белая «A»), бейджи (Б/У грейд, −%, НОВИНКА) — идентичны

### Тон и копирайт
- Простой, дружелюбный, по-деловому. Рус + Кыр. Валюта: «сом» (не символ).

## 2. Что ОТДАЁМ платформе (родные паттерны)

| Аспект | iOS (SwiftUI, HIG) | Android (Compose, Material 3) |
|---|---|---|
| Системный шрифт | SF Pro (та же типо-шкала) | Roboto / Golos Text |
| Навигация | таб-бар снизу, свайп-назад, NavigationStack | таб-бар (NavigationBar), системная «назад», предиктивный жест |
| Списки/скролл | bounce, pull-to-refresh (родной) | overscroll glow, SwipeRefresh |
| Кнопки/нажатия | лёгкое затемнение/масштаб | Material ripple |
| Диалоги/шторки | sheet, action sheet, alert | BottomSheet, Dialog (M3) |
| Пикеры (дата/время/выбор) | родные wheel-пикеры | родные Material-пикеры |
| Клавиатура/поля | родное поведение, типы клавиатур | то же |
| Уведомления | APNs, стиль iOS | FCM, каналы Android |
| Тактильность | Haptics (успех/ошибка) | Vibration/haptics |
| Иконки системных действий | SF Symbols | Material Symbols |

Правило: **контент и бренд — наши; «поведение оболочки» — системное.** Кнопка «В корзину»
выглядит одинаково (coral, радиус), но нажимается с родным откликом каждой ОС.

## 3. Компоненты → SwiftUI / Jetpack Compose

| Компонент | SwiftUI | Compose |
|---|---|---|
| Экран/фон | `ZStack` + `Color(bgLight)` | `Surface` + `Modifier.background` |
| Карточка | `RoundedRectangle(cornerRadius:16)` + shadow | `Card(shape=RoundedCornerShape(16.dp))` |
| Кнопка (primary) | `Button` + `.background(coral)` `.cornerRadius(12)` | `Button(colors=coral, shape=RoundedCornerShape(12.dp))` |
| Чип/бейдж | `Text().padding().background(capsule)` | `AssistChip` / кастом `Surface` |
| Таб-бар | `TabView` | `NavigationBar` |
| Список товаров | `LazyVGrid` | `LazyVerticalGrid` |
| Карусель категорий | `ScrollView(.horizontal)` + `LazyHStack` | `LazyRow` |
| Модалка/шторка | `.sheet` | `ModalBottomSheet` |
| Поле ввода | `TextField` + бренд-обводка | `OutlinedTextField` |
| Прогресс/лоадер | `ProgressView` (tint coral) | `CircularProgressIndicator` |
| Тост | кастом overlay (см. прототип) | `Snackbar` / кастом |

## 4. Состояния (обязательно на каждом экране — как в прототипах)
Empty / Loading (скелетоны) / Error / Success / Permission denied. Цвета берём из токенов
(danger для ошибок, success для успеха). Скелетоны — серый шиммер на `bgLight`.

## 5. Тёмная тема
- Витрина (Клиент) — светлая по умолчанию; поддержать системную тёмную (инверсия нейтралей, бренд-coral остаётся).
- POS / ERP / операционные — тёмные всегда (`bgDark` + `lime`-акцент), как в прототипах.

## 6. Доступность
- Контраст AA (проверить coral на белом для текста — использовать coralDeep для мелкого текста).
- Хиты нажатия ≥ 44×44 pt (iOS) / 48×48 dp (Android).
- Поддержка Dynamic Type / шкалирования шрифта.

## 7. Ассеты
- Логотип, иконки категорий, бейджи — единый набор, экспорт в SF Symbols-стиле + Material.
- Фото товаров — WebP/AVIF, адаптивные размеры, ленивая загрузка (см. оптимизацию в мастер-доке).
- `assets/p-*.png` в пакете — плейсхолдеры, заменить реальными.

---

**Итог:** пользователь на iOS и Android получает узнаваемый AliStore (цвет, карточки, логика),
но с родными навигацией, жестами и системными контролами. Обе команды сверяются с `.dc.html`
как с единым эталоном пикселей и поведения.

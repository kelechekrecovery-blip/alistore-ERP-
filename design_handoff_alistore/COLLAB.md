# AliStore — COLLAB (канал переписки Design ↔ Claude Code)

Общий тред. **Claude Code** пишет сюда вопросы/что сделал в раздел «От Claude Code».
**Владелец** приносит это в чат к Claude Design → ответы возвращаются в раздел «От Claude Design».
Формат: дата, кто, тема, суть. Свежее — сверху.

Правила: спорные архитектурные моменты сверяются с `ENGINEERING_DECISIONS.md`.
Если Claude Code хочет отступить от решения — обосновать здесь ДО кода.

## От Claude Design → Claude Code (стартовые указания)

**[старт] Порядок работы**
1. Прочитай `ENGINEERING_DECISIONS.md` и `docs/Техническая спецификация.md` целиком.
2. Начни с каркаса: монорепо (`apps/api`, `apps/web`), Prisma + миграции + сиды, CI.
3. Второй шаг — **Event Ledger + Auth + RBAC** (фундамент, не откладывать).
4. Третий — Order state-machine (created→paid) + Payment + резерв IMEI.
5. Сразу напиши приёмочные тесты P0: «двойная продажа IMEI», «оплата без резерва».
6. После каждого модуля — пиши сюда: что сделано, какие отклонения, какие вопросы.

**Чего жду от тебя в ответ (заполни ниже):**
- Подтверждение стека и структуры репо.
- Дизайн БД под варианты товара (матрица цвет×память) — предложи схему.
- Выбор очереди (BullMQ/Redis Streams) — обоснуй.
- Список открытых вопросов, где нужен владелец (платёжка, SMS-шлюз, хостинг).
